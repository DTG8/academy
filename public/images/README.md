# Images Directory

Place your images here! 

## Where to Put Images:

All images should be placed in this `public/images/` directory.

## Image Files Needed:

Based on the website structure, you'll need:

### Required Images:
1. **Hero Section** - Main hero/banner image
   - Suggested name: `hero.jpg` or `hero-image.jpg`

2. **Course Images** (5 images):
   - `web-development.jpg` - Web Development course
   - `devops.jpg` - DevOps course
   - `cloud-computing.jpg` - Cloud Computing course
   - `cybersecurity.jpg` - Cybersecurity course
   - `ai-ml.jpg` - AI/ML course

3. **Feature Images** (6 images, optional):
   - `real-time-learning.jpg` - Real-Time Learning feature
   - `personal-mentorship.jpg` - Personal Mentorship feature
   - `project-based-learning.jpg` - Project-Based Learning feature
   - `in-person-option.jpg` - In-Person Option feature
   - `career-growth.jpg` - Career Growth feature
   - `latest-curriculum.jpg` - Latest Curriculum feature

4. **CTA Section** - Background image (optional)
   - Suggested name: `cta-background.jpg`

## Supported Formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.svg`

## Notes:
- Use descriptive filenames (lowercase with hyphens)
- Recommended image sizes:
  - Hero: 1200x800px or larger
  - Course cards: 800x500px
  - Feature cards: 600x300px
  - CTA background: 1920x1080px (optional, can be smaller)

Once you place the images here, tell me the filenames and I'll update the code to use them!
